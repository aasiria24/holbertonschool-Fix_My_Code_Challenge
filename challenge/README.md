# Fix My Code Challenge

## Description
A collection of buggy code snippets in different languages, fixed without recoding from scratch.

## Technologies
- Python
- JavaScript (Node.js)
- Ruby
- C

## Files
| File | Language | Bug Fixed |
|---|---|---|
| `0-fizzbuzz.py` | Python | FizzBuzz condition order |
| `1-print_square.js` | JavaScript | parseInt radix 16 → 10 |
| `2-sort.rb` | Ruby | insert index i-1 → i |
| `3-user.py` | Python | password hashing & comparison |
| `4-delete_dnodeint/` | C | doubly linked list pointer relinking |
